## ğŸ”§ Wiki Setup

The glossary pages are in the `wiki/` folder. To enable the GitHub wiki:

2. Create the first page (Home) with any content
3. Clone the wiki: `git clone https://github.com/tanakasan3/nihongo-glossary.wiki.git`
4. Copy the files: `cp wiki/*.md nihongo-glossary.wiki/`
5. Push: `cd nihongo-glossary.wiki && git add -A && git commit -m "Import glossary" && git push`

Or just browse the `wiki/` folder directly in this repo!
